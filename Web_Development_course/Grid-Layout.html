CSS Grid is 2-dimensional system for creating web layouts. Grid allows arrangement of items into rows and columns.

An HTML element becomes a grid container when its display property is set to grid or inline-grid.

Items are arranged in a grid and have a minimium width, also they adjusts to screen size, making sure that grid items fit as many as possible rows.
