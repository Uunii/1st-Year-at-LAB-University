Introduction of HTML

HTML (HyperText Markup Language) is the standard language for creating web pages. A browser reads HTML file and renders the content based on the tags.

A HTML page requires following tags to stack:

(HTML)

Creates an HTML document

(BODY)

Body is a container element for all the visible contents, such as headings, paragraphs, images,

(HEAD)

HTML headings are defined with the (h1) to (h6) tags.
Numbers after (h) defines the importance of the heading with different sizes h1 being largest and h6 being smallest.

(P)

HTML paragraphs are defined with the (p) tag

Other basic tags that can be used in html page:

(IMG)

The (img) tag is used to embed an image in an HTML page. The (src) attribute specifies the path to the image to be displayed:

(A)

The (a) tag defines a hyperlink. The (href) attribute specifies the URL of the page the link goes to:
